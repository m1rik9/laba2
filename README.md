### Задание 01: Количество кодовых слов

**Задание:** Олег составляет таблицу кодовых слов для передачи сообщений, каждому сообщению соответствует своё кодовое слово. В качестве кодовых слов Олег использует 4-буквенные слова, в которых есть только буквы A, B, C, D, E, X, Z, причём буквы X и Z встречаются только на двух первых позициях, а буквы A, B, C, D, E  — только на двух последних. Сколько различных кодовых слов может использовать Олег?

**Решение:** Использовано комбинаторное правило умножения. Для первых двух позиций доступно 2 буквы (X и Z), для последних двух — 5 букв (A, B, C, D, E). Общее количество комбинаций: 2 × 2 × 5 × 5 = 100. Для проверки использована функция `itertools.product()`, генерирующая все возможные комбинации.

```python
from itertools import product

def count_code_words():
    """
    Подсчет количества 4-буквенных кодовых слов,
    где на первых двух позициях только X или Z,
    а на последних двух только A, B, C, D, E.
    """
    first_positions = ['X', 'Z']
    last_positions = ['A', 'B', 'C', 'D', 'E']
    
    # Генерируем все возможные комбинации
    words = list(product(first_positions, first_positions, last_positions, last_positions))
    
    return len(words)

# Вызов функции
result1 = count_code_words()
print(f"Задача 1: {result1}")
```
